## 🌟 Overview

The Hackathon Management System is a full-stack web application designed to simplify the management of hackathons for organizers, participants, and judges.

This platform allows:
- Participants to register and submit projects
- Organizers to manage hackathons efficiently
- Judges to evaluate projects and declare results

---

## ✨ Key Features

✅ User Authentication & Authorization  
✅ Participant Dashboard  
✅ Organizer Dashboard  
✅ Judge Evaluation Panel  
✅ Project Submission System  
✅ Result & Scoreboard Management  
✅ Query & Response System  
✅ Responsive Modern UI  
✅ Dark Mode Support

## ⚙️ Installation Guide

### Clone Repository

```bash
git clone https://github.com/BuliyaRathore/hackathon-management-system.git
```

### Open Project Folder

```bash
cd hackathon-management-system
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm start
```
